# Routing Sheet Creation

## Purpose
Create a routing sheet for every tax return to track its progress through the preparation process and ensure all necessary information is captured.

## When to Create
- Client drop-off
- Portal document upload (auto-generated)
- In-person appointment
- Documents received by mail

## Routing Sheet Fields

## Digital Routing Sheet
If using Tax Helper system:
1. Create return in system
2. System auto-generates routing sheet
3. Update with received documents
4. Assign to preparer

## Physical Routing Sheet
1. Print blank routing sheet
2. Complete all sections
3. Attach to document folder
4. Place in preparer bin

## Status Tracking
Update status in CCH/TaxHelper as return progresses:
1. Intake
2. Documents Pending
3. Documents Complete
4. In Preparation
5. Review Needed
6. Ready for Signing
7. Completed
8. Filed

## Notes
- One routing sheet per tax return
- Update status same day as changes
- Routing sheet stays with return until filed
