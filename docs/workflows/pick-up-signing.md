# Client Pick-Up & Signing Workflow

## Purpose
Ensure clients properly review, sign, and receive their completed tax returns with all necessary documentation.

## Pre-Arrival Preparation

### Day Before
1. Confirm pick-up appointment
2. Send reminder (email/text)
3. Prepare return package

### Return Package Contents
- [ ] Tax return copy (client copy)
- [ ] E-file authorization forms
- [ ] Payment vouchers (if applicable)
- [ ] State return copy
- [ ] Invoice
- [ ] Tax planning notes (if any)

## Client Arrival

### 1. Greeting
- Welcome client by name
- Confirm purpose of visit
- Escort to meeting area or counter

### 2. Return Review
Present return summary:
- Filing status
- Total income
- Adjusted gross income
- Refund amount OR amount owed
- Key changes from prior year

**Important:** Allow client to ask questions before signing.

### 3. Signature Collection

**E-File Returns:**
- [ ] Form 8879 (IRS e-file authorization)
- [ ] State e-file authorization (if applicable)

**Paper Returns:**
- [ ] Signature page of Form 1040
- [ ] Any additional signature lines

**DocuSign (if remote):**
1. Send e-file forms via DocuSign
2. Wait for client completion
3. Download signed copies
4. File in client record

### 4. Refund Options
Discuss refund delivery:
- Direct Deposit (verify bank info)
- Paper Check (allow 6-8 weeks)
- Refund Transfer (explain fees)

If Balance Due:
- Payment due date
- Payment options (direct debit, check, card)
- Voucher instructions
- Penalty/interest warnings

### 5. Document Delivery
Provide to client:
- Copy of return (paper or electronic)
- Signed e-file authorization copy
- Payment vouchers
- Extension forms (if applicable)
- Next year's tax organizer

### 6. Payment Collection
If paying at pick-up:
- Process payment (see Payment Processing)
- Provide receipt
- Update CCH/AR

If billing:
- Review invoice
- Confirm payment method/timeline
- Update CCH with arrangement

### 7. Filing Confirmation
Explain next steps:
- "Your return will be e-filed within 24-48 hours"
- "You'll receive IRS/State acceptance confirmation by email"
- "Keep your documents for at least 3 years"

## Post-Appointment

### Immediate Tasks
1. Update CCH status to "Ready for Filing"
2. Queue return for e-file
3. File signed authorization forms
4. Process any payments

### Follow-up
- E-file within 24-48 hours
- Send acceptance confirmation
- Follow up on any outstanding payments

## Return of Documents
Ask client about original documents:
- Return all original documents (recommended)
- Shred documents (with client permission)
- Keep copies in file

## Special Situations

### Client Has Questions
- Answer what you can
- Schedule callback with preparer if needed
- Never guess or make up answers

### Client Disagrees with Return
- Listen to concerns
- Note specific issues
- Schedule meeting with preparer
- Do NOT have client sign if unhappy

### Client Cannot Come In
Options:
- DocuSign (electronic signature)
- Mail return with signature page
- Authorized representative pick-up

## Notes
- Never pressure client to sign
- Allow time for questions
- Confirm contact information is current
- Thank client for their business
