# Start of Day Procedures

## Purpose
Ensure the office is properly set up and ready for client interactions at the beginning of each business day.

## Timing
Begin procedures at 8:00 AM, 30 minutes before first appointments.

## Checklist

### 1. Office Setup
- [ ] Unlock front door
- [ ] Turn on lights and HVAC
- [ ] Start all computers
- [ ] Check printer status and paper supply
- [ ] Ensure copier/scanner is operational

### 2. System Login
- [ ] Log into CCH Practice Management
- [ ] Log into Drake Tax Software
- [ ] Log into Tax Helper dashboard
- [ ] Open email and client portal dashboard
- [ ] Check voicemails and note urgent messages

### 3. Daily Preparation
- [ ] Print daily appointment list from CCH
- [ ] Pull files for scheduled appointments
- [ ] Prepare routing sheets for any expected drop-offs
- [ ] Stock front desk with:
  - Tax organizers
  - Engagement letters
  - Payment forms
  - Business cards

### 4. Communication Check
- [ ] Review overnight portal uploads
- [ ] Check for urgent client emails
- [ ] Identify any returns ready for pick-up
- [ ] Note any payments due today

### 5. Team Coordination
- [ ] Brief team on day's schedule
- [ ] Identify priority returns
- [ ] Discuss any special client situations
- [ ] Confirm preparer availability

## Notes
- First client appointment typically at 9:00 AM
- Allow extra time during tax season (January - April)
- Keep emergency contact list accessible
