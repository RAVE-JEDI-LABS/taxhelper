# End of Day Procedures

## Purpose
Ensure all daily tasks are completed, systems are secured, and the office is ready for the next business day.

## Timing
Begin 30 minutes before closing (typically 4:30 PM)

## Checklist

### 1. Client Follow-up
- [ ] Return all voicemails
- [ ] Respond to urgent emails
- [ ] Send any pending client communications
- [ ] Update CCH notes for all client interactions

### 2. Document Processing
- [ ] All drop-off documents scanned
- [ ] Tax Helper OCR queue processed
- [ ] Physical documents in preparer bins
- [ ] Portal uploads reviewed and logged

### 3. Financial Reconciliation
- [ ] Count cash drawer
- [ ] Total checks received
- [ ] Print Square daily report
- [ ] Verify totals match CCH entries
- [ ] Prepare bank deposit
- [ ] Complete daily deposit slip
- [ ] Secure or deposit funds

### 4. System Updates
- [ ] Update all return statuses in CCH
- [ ] Mark completed appointments
- [ ] Review tomorrow's appointment list
- [ ] Pull files for tomorrow's appointments
- [ ] Update any pending tasks

### 5. Office Security
- [ ] Lock all file cabinets
- [ ] Secure sensitive documents
- [ ] Log out of all systems:
  - CCH
  - Drake
  - Tax Helper
  - Email
  - Client Portal
- [ ] Clear desk of client information
- [ ] Shred any sensitive waste

### 6. Physical Closing
- [ ] Turn off computers and monitors
- [ ] Turn off copier/scanner
- [ ] Check that printer is off
- [ ] Adjust thermostat
- [ ] Turn off lights
- [ ] Lock front door
- [ ] Set alarm (if applicable)

## Emergency Checklist
Before leaving, ensure:
- [ ] Emergency contact list is posted
- [ ] After-hours voicemail is active
- [ ] No client documents left unsecured
- [ ] No active credit card terminals

## During Tax Season (Jan-Apr)
Additional tasks:
- [ ] Update extension tracking list
- [ ] Review deadline calendar
- [ ] Communicate any urgent items to preparers
- [ ] Plan next day's priorities

## Notes
- Never leave client documents unsecured overnight
- Report any security concerns immediately
- Complete checklist even if leaving early
- Last person out is responsible for full closing
