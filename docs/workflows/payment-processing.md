# Payment Processing Workflow

## Purpose
Process client payments accurately and efficiently using the appropriate payment method.

## Accepted Payment Methods
1. Cash
2. Check
3. Credit/Debit Card (Square)
4. Payment Plan

---

## Cash Payments

### Procedure
1. Count cash in front of client
2. Confirm amount verbally
3. Issue receipt from receipt book
4. Record in daily cash log:
   - Client name
   - Amount
   - Invoice number
   - Date/time
5. Store cash in secure location

### End of Day
- Count cash drawer
- Verify against daily log
- Prepare bank deposit slip
- Secure overnight or deposit same day

---

## Check Payments

### Procedure
1. Verify check is properly completed:
   - Date
   - Payee (Gordon Ulen CPA)
   - Amount (numeric and written)
   - Signature
2. Stamp back "For Deposit Only - Gordon Ulen CPA"
3. Log in check register:
   - Check number
   - Bank name
   - Amount
   - Client name
4. Issue receipt
5. Prepare for bank deposit

---

## Credit/Debit Card (Square)

### Procedure
1. Open Square Point of Sale app
2. Select "Charge Amount"
3. Enter invoice amount
4. Hand terminal to client or tap/insert card
5. Wait for approval
6. Email or print receipt
7. Verify transaction appears in Square dashboard

### Payment Plans via Square
1. Discuss payment options with client
2. Set up recurring payment in Square:
   - Navigate to Invoices
   - Create recurring invoice
   - Set payment schedule
3. Document agreement in CCH
4. Provide client copy of agreement

---

## Recording in CCH

After any payment:
1. Open client record in CCH
2. Navigate to Accounts Receivable
3. Record payment:
   - Amount
   - Date
   - Method
   - Reference number (check # or Square transaction ID)
4. Update invoice status

---

## Refunds

If refund is needed:
1. Get manager approval
2. For card payments:
   - Process refund through Square
   - Keep transaction record
3. For cash/check:
   - Issue company check
   - Get two signatures
4. Document reason in CCH

---

## End of Day Reconciliation

1. Print daily payment report from CCH
2. Count cash drawer
3. Total checks received
4. Print Square daily summary
5. Verify totals match
6. Report any discrepancies immediately
7. Prepare bank deposit
8. Secure all funds

## Notes
- Never leave payments unattended
- Report discrepancies same day
- Keep payment area secure
- Shred any documents with card numbers
